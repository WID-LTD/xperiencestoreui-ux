# Integrations

This directory contains the integration modules for external services.

## Maps

- TomTom Map API: `maps/tomtom.js`

## Social Media & Auth

- Facebook API: `social/facebook.js`
- GitHub API: `social/github.js`
- LinkedIn API: `social/linkedin.js`
- X (Twitter) API: `social/x.js`
- Google/YouTube API: `social/google.js`

## Communication

- Brevo: `email/brevo.js`
- Mailgun: `email/mailgun.js`

## Translation

- SimplyTranslate.ai: `translation/simplytranslate.js`

## Payment Gateways

- PayPal: `payment/paypal.js`
- Payoneer: `payment/payoneer.js`
- Paystack: `payment/paystack.js`
- Flutterwave: `payment/flutterwave.js`
- Klasha: `payment/klasha.js`
- Coinbase Commerce: `payment/coinbase.js`
- WorldFirst: `payment/worldfirst.js`

## Usage

Import these modules in the respective controllers to use the services. Ensure API keys are set in `.env`.
